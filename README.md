Proyecto desarrollado en Python utilizando la biblioteca Pandas para demostrar el proceso básico de análisis y preparación de datos. El proyecto muestra cómo cargar información, limpiar datos incompletos, transformarlos y obtener estadísticas descriptivas de manera sencilla.

📖 Descripción

En cualquier proyecto de análisis de datos, la información suele contener valores faltantes o inconsistencias que deben corregirse antes de realizar un análisis.

Este proyecto implementa un flujo básico de procesamiento de datos que incluye:

Creación de un DataFrame.
Limpieza de datos.
Reemplazo de valores faltantes.
Transformación de datos.
Clasificación de información.
Obtención de estadísticas descriptivas.

El ejemplo utiliza un conjunto de datos con calificaciones de estudiantes para ilustrar cada uno de estos procesos.
